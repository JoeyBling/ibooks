## 本地预览
```bash
npm install
npm run dev
```

## 编译部署
```bash
npm run clean
npm run dist
//生成的public文件夹的内容上传至GitHub/其他服务器，或者使用 hexo d 进行配置自动部署
```

## 执行压缩
```bash
npm run gulp
```

1. 崇尚简约

3. 希望把加载速度做到极致（努力中）
4. 让大家把注意力放到内容上。这是本主题设计初衷
5. 主题不支持IE6，7，8。以后也不会
